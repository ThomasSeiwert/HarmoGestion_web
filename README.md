<h1>HarmoGestion_web</h1>
<h2>Introduction</h2>
<p>Ceci est un projet éducatif au sein de l'AFPA de Pompey. Il permet de gérer une association de musique qui gère des membre, une liste d'instruments de musique, des cours de musique et des représentations (cpncerts). Les technologies utilisées sont:
<br/>•Java 25
<br/>•HTML5
<br/>•CSS3
<br/>•JavaScript
<br/>•Javadoc (exclue des commits)
<br/>•SpringBoot
<br/>•Selenium
<br/>•BootStrap
<br/>•Hibernate
<br/>•Maven
<br/>•NPM
<br/>•Git/GitHub
</p><br/>

<h2>Conception</h2>
<p>Le dossier de conception contient le document SCRUM, le dossier de déploiement, les compte-rendu de réunion, les diagrammes de classe, de use-case, l'enchainement des écrans, le MCD Looping, les plan de test et la charte graphique. Il y figure également le script de génération de base de données pour MySQL.</p><br/>

<h2>Utilisation</h2>
<p>Le projet HarmoGestion_web s'utilise en complément de HarmoGestion_API (https://github.com/YamashiFenikkusu/HarmoGestion_API) et d'une base de données MySQL fonctionnelle d'après les instructions fournie dans le dossier de déploiement (script fourni tel que mentionné dans le précédent paragraphe).</p><br/>

<h2>Remarques supplémentaires</h2>
<p>•Certaines pages HTML et fragments Thymeleaf du projet utilisent les capacités du fichier messages.properties Thymeleaf.
<br/>•L'application web utilise le port 9001, et l'API le port 9002.
<br/>•Les configurations de l'application web et de l'API s'effectuent dans le fichier application.properties (procédure détaillée dans le dossier de déploiement).</p>
